## Solution 1 (20 marks)

**Question:** Define Computer Science and explain its major areas of study.

**Answer:**
Computer Science is the scientific and practical approach to computation and computational systems. It uses mathematical and engineering principles to design and develop computer systems and software applications.

### Major Areas:

1. **Programming & Software Engineering:** Software development methodologies, programming paradigms
2. **Data Structures & Algorithms:** Efficient data organization and problem-solving techniques
3. **Computer Systems:** Hardware-software interaction, operating systems
4. **Database Systems:** Data storage, retrieval, and management
5. **Computer Networks:** Network protocols, distributed systems
6. **Artificial Intelligence:** Machine learning, expert systems
7. **Human-Computer Interaction:** User interface design, usability

## Solution 2 (25 marks)

### a) Algorithm Definition (12 marks)

**Algorithm** is a finite set of well-defined instructions that provides a step-by-step procedure to solve a specific problem.

**Example:**

```
Algorithm: Find Maximum of Two Numbers
Input: Two numbers a, b
Output: Maximum number

Step 1: Start
Step 2: Read a, b
Step 3: If a > b then max = a
Step 4: Else max = b
Step 5: Print max
Step 6: Stop
```

### b) Algorithm vs Program (13 marks)

| Algorithm              | Program                     |
| ---------------------- | --------------------------- |
| Problem-solving method | Implementation of algorithm |
| Language independent   | Language specific           |
| Mathematical notation  | Specific syntax             |
| Human readable         | Machine executable          |

## Solution 3 (25 marks)

### Data Structure (8 marks)

**Data Structure** is a systematic way of organizing and storing data that provides efficient access and modification capabilities.

**Examples:** Array, Linked List, Stack, Queue, Tree

### Programming Language (8 marks)

**Programming Language** is a formal language used for writing computer instructions.

**Examples:** C, Java, Python, JavaScript

### Operating System (9 marks)

**Operating System** is system software that manages computer hardware and software resources and provides common services for computer programs.

**Examples:** Windows, Linux, macOS, Android

## Solution 4 (30 marks)

### a) Evolution of Computers (15 marks)

1. **First Generation (1940-1956):** Vacuum tubes, ENIAC, UNIVAC
2. **Second Generation (1956-1963):** Transistors, smaller size, COBOL, FORTRAN
3. **Third Generation (1964-1971):** Integrated Circuits, minicomputers
4. **Fourth Generation (1971-present):** Microprocessors, personal computers
5. **Fifth Generation (present-future):** Artificial Intelligence, quantum computing

### b) Key Components (15 marks)

1. **Input Devices:** Keyboard, mouse, scanner
2. **Processing Unit (CPU):** Control unit, ALU, registers
3. **Memory:** Primary (RAM, ROM), Secondary (HDD, SSD)
4. **Output Devices:** Monitor, printer, speakers
5. **System Bus:** Data bus, address bus, control bus
